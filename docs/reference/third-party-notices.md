# Third-Party Notices

Roomie Kit uses third-party open-source packages and service SDKs, including React, Vite, Supabase, Agora, Babylon.js, Capacitor, and supporting libraries.

Before releasing your fork:

- Review dependency licenses.
- Keep required notices.
- Check service SDK terms for your use case.
- Document any third-party assets you add.

The repository-level `THIRD_PARTY_NOTICES.md` should be kept up to date with release changes.
