# Asset Licenses

Roomie-owned starter assets are intended to be redistribution-safe with the public starter.

Before publishing a fork:

- Confirm every bundled image, GLB, audio file, WASM file, and texture is owned or licensed for redistribution.
- Keep large binary assets in Git LFS or a documented release asset bundle.
- Update `ASSET_LICENSES.md` when replacing bundled assets.
- Preserve notices for any third-party code or assets.

See [Asset folders](../assets.md) for packaging strategy.
