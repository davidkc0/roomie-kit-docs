# What You Can Build

Roomie Kit can be used for:

- Avatar-based virtual rooms with profiles, presence, furniture, and movement.
- Voice rooms and video chat with Agora.
- Theater-style livestreams.
- Personal room group video.
- Direct 1:1 calls.
- Standalone livestreaming or video-chat products without virtual spaces.
- iOS and Android apps through Capacitor.
- Optional product modules such as waitlist, invites, economy, gifts, push, and payments.

## Product Shapes

Use the full starter if you want a social world with rooms, avatars, games, and realtime communication.

Use the streaming layer only if you want a lighter product such as:

- Creator livestreaming.
- Webinar rooms.
- Private group video.
- Embedded video chat.
- A watch-party theater.

Roomie Kit Cloud can deploy either path; the default onboarding preset starts with the full starter.
