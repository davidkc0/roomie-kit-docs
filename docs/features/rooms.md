# Rooms

Roomie Kit ships an avatar-room experience you can use as the main product or as one module inside a larger app.

Built in:

- Realtime room presence and membership through Supabase.
- Avatar movement inside Babylon.js spaces.
- Room voice and TV-head video through the shared media provider.
- Starter room models, furniture hooks, floor/wall assets, and local override folders.
- Mobile-ready controls through Capacitor wrappers.

Room assets use the same canonical keys as the starter bundle, so replacing a room can be as simple as adding a matching file under `web/public/roomie-local/rooms`.
