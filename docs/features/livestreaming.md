# Livestreaming

Roomie Kit includes livestreaming as a first-class media path, not just as part of the 3D room.

Built in:

- Theater-style go-live and watch flows.
- Channel-scoped Agora tokens from Supabase Edge Functions.
- Deterministic channel names for stream sessions.
- Shared `MediaProvider` and `MediaSession` controls for join, leave, publish, subscribe, cleanup, and reconnect handling.
- A streaming-only path for products that do not need virtual spaces.

Use this path for creator rooms, watch parties, webinars, events, or embedded live video products.
