# Roomie Kit

Roomie Kit is a modular starter for realtime social products. It includes a Vite/React app, Supabase backend, Agora media layer, Babylon.js virtual rooms, Capacitor mobile wrappers, starter assets, and customization hooks.

Use it as a full virtual-room app, or use the media layer as a livestreaming and video-chat foundation without the 3D spaces.

## Roomie Kit Cloud

Roomie Kit Cloud is the paid path for teams that want browser-login setup, guided deployment, health checks, starter asset hosting, Codex/Claude Code plugin access, update tooling, and support. See the [plans section](https://roomiekit.io/#pricing) for current pricing, then follow [CLI onboarding](managed-hosting/onboarding.md).
