# Support And Cancellation

For support, email `support@roomiekit.io`.

Include:

- Account email.
- Deployed URL.
- Supabase project ref if relevant.
- The CLI command that failed.
- Error output with secrets removed.

Do not send Supabase service-role keys, database passwords, Vercel tokens, Agora certificates, or Roomie license keys in support messages.

## Cancellation

Roomie Kit Cloud can be canceled from the account dashboard billing portal or by contacting support before the next billing period.

Cancellation stops future renewals. Access to managed CLI entitlements, update tooling, and support may end when the subscription becomes inactive.
