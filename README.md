# Roomie Kit Docs

This repo contains the GitBook source for [docs.roomiekit.io](https://docs.roomiekit.io).

The public starter app lives in [`davidkc0/roomie-kit`](https://github.com/davidkc0/roomie-kit). Keeping docs in this separate repo keeps starter clones focused on app code.

## Structure

- GitBook config: [`.gitbook.yaml`](.gitbook.yaml)
- GitBook landing page: [`docs/README.md`](docs/README.md)
- GitBook sidebar: [`docs/SUMMARY.md`](docs/SUMMARY.md)

## Local Editing

Edit Markdown under `docs/`, then push this repo. GitBook should sync from this repository once the GitBook space is pointed at `davidkc0/roomie-kit-docs`.
